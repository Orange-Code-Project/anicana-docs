Detail API
=======================================

| /detail
| categoryにトークンコントラクトアドレス、tokenIdにトークンIDを指定するとトークンの詳細データを取得できる。


Ipfs Upload  API
=======================================

| /up_ipfs\
| IPFSにファイルをアップロードすることができる。アップロードしたHashを取得できる。
