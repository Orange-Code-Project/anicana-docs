コンセンサスアルゴリズム¶

アニカナのコンセンサスアルゴリズムは、IBFT(Istanbul Byzantine Fault Tolerance)を採用する。 ブロックの生成間隔は5秒とする。IBFTで生成されたブロックは、ブロックの提案ノードと複数の Validatorによって署名され、このブロックを改竄するためのはそれらのすべての秘密鍵を知る必要 があるため非常に高い改竄耐性を備えている。
